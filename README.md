# КнигоМир — Профессиональный Книжный Магазин (MPA Edition)

Современное веб-приложение для продажи книг, реализованное на классической многостраничной архитектуре (MPA) с использованием статических HTML-файлов и динамического взаимодействия через API.

## 🚀 Архитектура проекта

Приложение переведено с шаблонизатора EJS на **чистую MPA-архитектуру**. Это обеспечивает высокую скорость работы, простоту деплоя и четкое разделение ответственности (Separation of Concerns).

### Основные принципы:
*   **Static Entry Points:** Каждая страница сайта — это отдельный физический `.html` файл в корне проекта.
*   **Client-Side Rendering (CSR):** Динамические данные (список книг, фильтры, корзина, профиль) подгружаются на стороне клиента через API-запросы (Axios) и вставляются в DOM с помощью JavaScript (`app.js`).
*   **State Management:** Авторизация и состояние корзины управляются через JWT (хранится в `localStorage`) и синхронизируются с сервером.
*   **REST API Backend:** Сервер на Express.js служит исключительно для предоставления данных и обработки бизнес-логики.

## 📁 Структура проекта

```text
├── config/             # Конфигурация БД (Sequelize)
├── middleware/         # Защита маршрутов (Auth, Admin)
├── models/             # Модели данных Sequelize (User, Book, Order, Review, Cart)
├── routes/             
│   ├── api.js          # Основное API (Корзина, Заказы, Отзывы)
│   ├── auth.js         # API авторизации и регистрации
│   ├── books.js        # API для работы с каталогом книг
│   └── views.js        # Роутинг страниц (отдает HTML-файлы)
├── server.js           # Точка входа бэкенда, сидинг данных и запуск сервера
├── app.js           # Главный скрипт фронтенда (логика всех страниц)
├── index.html          # Главная страница
├── catalog.html        # Страница каталога с фильтрами
├── book.html           # Страница описания книги
├── cart.html           # Корзина покупок
├── profile.html        # Личный кабинет пользователя
├── auth.html           # Страница входа и регистрации
├── admin.html          # Админ-панель
├── style.css           # Глобальные стили и темы
└── .env                # Настройки окружения (БД, JWT Secret)
```

## 💻 Стек технологий

*   **Frontend:** HTML5 (Semantic), CSS3 (Custom Variables, Dark Theme), Vanilla JavaScript (ES6+), Axios.
*   **Backend:** Node.js, Express.js (v5).
*   **Database:** PostgreSQL + Sequelize ORM.
*   **Security:** JSON Web Tokens (JWT), Bcryptjs (хэширование паролей).

## 🛠 Функциональные возможности

### 🏠 Главная страница (`index.html`)
*   Герой-секция с динамическими статами.
*   Блок "Популярное сейчас", автоматически подгружающий книги с самым высоким рейтингом.
*   Интерактивные элементы и плавная анимация.

### 📚 Каталог и Поиск (`catalog.html`)
*   **Умные фильтры:** По жанрам, авторам и диапазону цен.
*   **Сортировка:** По популярности, цене (возр/убыв), рейтингу и новизне.
*   **Живой поиск:** Поиск по названию, автору или ISBN с выпадающими подсказками в шапке.
*   **Пагинация:** Постраничный вывод товаров (по умолчанию 12 книг на страницу).

### 📖 Страница книги (`book.html`)
*   Полная информация о книге: обложка, описание, характеристики (ISBN, страницы, издательство).
*   **Система отзывов:** Просмотр и добавление отзывов с рейтингом (доступно авторизованным пользователям).
*   Автоматический пересчет рейтинга книги при добавлении новых отзывов.

### 🛒 Корзина (`cart.html`)
*   Управление списком покупок: добавление, удаление, изменение количества.
*   Мгновенный пересчет итоговой суммы.
*   Оформление заказа в один клик с автоматической очисткой корзины.

### 👤 Профиль (`profile.html`)
*   Информация о пользователе.
*   История заказов с актуальными статусами.
*   Синхронизация состояния между вкладками.

### 🛠 Админ-панель (`admin.html`)
*   Доступна только пользователям с ролью `admin`.
*   **CRUD книг:** Добавление новых поступлений, редактирование существующих, удаление.
*   **Мониторинг:** Просмотр всех заказов и управление отзывами пользователей.

## ⚙️ Установка и запуск

1.  **Настройка БД:** Создайте базу данных в PostgreSQL.
2.  **Конфигурация:** Создайте `.env` файл на основе примера:
    ```env
    DB_NAME=bookstore
    DB_USER=postgres
    DB_PASS=your_password
    DB_HOST=localhost
    JWT_SECRET=your_super_secret_key
    PORT=3000
    ```
3.  **Установка зависимостей:**
    ```bash
    npm install
    ```
4.  **Запуск:**
    ```bash
    npm start
    ```
    *При первом запуске сервер автоматически наполнит базу данных 35 книгами для теста.*
